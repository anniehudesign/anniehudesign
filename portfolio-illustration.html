Skyscape

A series of enclosed skyscapes created as a passion project to improve Illustrator skills. Each indicates different times of the day, showing the changes of the sky through time.

Davis Accounting Society Icons (In Progress)

This was a series of icons created for the Davis Accounting Society (DAS) to facilitate powerpoints used at the club's meetings. More to be added as more icons are created.

Coffee Icons (In Progress)

A series of icons created to express the different methods and ingredients needed to create coffee. More to be added!

Color Theory

The goal of this project was to create a collection of illustrations that utilizes different aspects of the color theory. In this project, an illustration of a terrarium was created in order to fully utilize the color palette through the light and shadow as well as the reflective properties of the glass. The color themes used in the project were analogous,complementary, monochrome, and triad.
